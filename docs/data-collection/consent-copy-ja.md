# データ利用の同意文（案）

最終更新日: 2026-03-08  
版: `2026-03-ja-v1`

## 1. 取得するデータ
- 入力データ: 収入、固定費、支出、納得度、価値観カテゴリ
- プロフィールデータ: 年齢、世帯人数、年収帯、住居形態、地域など
- 利用データ: 画面操作や利用状況（機能改善のため）

## 2. 利用目的
- 家計スコア（家計納得度・家計安定度）の算出
- 同属性比較やベンチマーク機能の提供
- アプリ機能改善と品質向上

## 3. 取り扱い方針
- 比較機能では、個人を特定できない集計単位でのみ表示します。
- サンプル数が少ない属性グループは、比較表示を行いません。
- 分析は匿名IDを用いて実施し、氏名・メール等の個人情報は必須にしません。

## 4. 同意と撤回
- 同意はいつでも設定画面から撤回できます。
- 撤回後は、新規データ収集を停止します。
- 既存データの削除を希望する場合は、削除申請を行えます。

## 5. 連絡先
- 運営問い合わせ: support@example.com
